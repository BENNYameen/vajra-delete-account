Delete Account Request

Users can request deletion of their Vajra EV Charging account and associated data.

How to Request Account Deletion

  1. Open the Vajra EV Charging app
  2. Navigate to Profile → Delete Account
  3. Submit the deletion request

Alternatively, users may request account deletion by emailing:

bennyamenasirvatham@gmail.com

Data Deleted

  • User profile information
  • Authentication data
  • Charging session history
  • Wallet-related user data

Data Retention

Certain transaction records may be retained temporarily to comply with legal and financial obligations.
